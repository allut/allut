## Featured Project

### [claude-anvil](https://github.com/allut/claude-anvil)

> Evidence-first code review plugin for Claude Code.

Claude writes code and claims "build passed ✅" — without running a single command. `claude-anvil` makes that impossible: every verification step is a real tool call with an exit code, recorded in a SQLite audit ledger.

```
/plugin marketplace add allut/claude-anvil
/plugin install claude-anvil@allut-claude-anvil
/claude-anvil:anvil-setup
```

Ported from [burkeholland/anvil](https://github.com/burkeholland/anvil) (originally for GitHub Copilot CLI).
